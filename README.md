# Task Manager 

A simple and efficient **Task Manager Web Application** that helps users organize daily tasks by creating, updating, and tracking their progress. This project focuses on clean UI design and basic task management functionality using frontend technologies.

---

## Features

- Add new tasks  
- Edit existing tasks  
- Delete tasks  
- Mark tasks as completed or pending  
- Simple and user-friendly interface  
- Responsive layout (basic)

---

## Technologies Used

- HTML  
- CSS  
- JavaScript  

---

## Project Objective

The objective of this project is to build a practical web application to improve productivity through task organization while strengthening frontend development skills and logical workflow design.
